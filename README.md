# Famine Database

This dataset compiles historical famine events with associated drivers, impacts, and contextual notes.

**File:** famine_database.xlsx

**Columns include:**
- Region / Location
- Start Year / End Year
- Primary Drivers (climate, conflict, market failure, disease, governance)
- Early Signals (e.g., crop failure, livestock stress, price spikes)
- Mortality / Migration Notes
- Sources (citation or archive reference)

This dataset is a work in progress and will expand as additional events and document sources are reviewed.
